Atividades - Tags de Texto

Nesta atividade, você aprenderá a utilizar tags de texto para formatar o conteúdo em uma página web

A seguir, temos um exemplo de trecho de código em HTML para enfatizar um texto:

<strong>Deixa o texto em negrito (forma mais recomendada)</strong>, ou <b>Texto em negrito</b> → Texto enfatizado

<em> Deixa o texto em itálico (forma mais recomendada)</em>, ou <i>Deixa o texto emitálico</i> → Texto enfatizado
<small> Deixa o texto menor </small> → Texto enfatizado
<u>Texto sublinhado</u> → Texto enfatizado
<u><em>Texto sublinhado e em Itálico</em></u> → Texto enfatizado

Experimente utilizar outras tags de texto, como <strong>, <u>, <sub>, <sup>, entre outras, para ver como elas alteram a aparência do texto
